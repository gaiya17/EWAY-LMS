📘 Eway Institute Learning Management System (LMS)


🚀 Project Overview

The Eway Learning Management System (LMS) is a web-based platform designed to digitize and automate the operations of an educational institute.
Currently, the institute uses tools like WhatsApp, Google Drive, and manual payments, which creates problems like:
Data loss
Time waste
Poor communication
No proper tracking system

👉 This system solves all those issues by providing a centralized platform where:

Students
Teachers
Admins
Institute Officers

can manage everything in one place.

🎯 Main Features


👨‍🎓 Student Features

Register & login securely
Browse and enroll in courses
Online & offline payment support
Attend online & physical classes
Access notes, videos, and materials
Submit assignments & quizzes
View grades and feedback
Track attendance (QR-based system)
Chat with teachers
Get notifications & announcements

👨‍🏫 Teacher Features

Create and manage courses
Upload study materials (videos, notes, quizzes)
Schedule classes (online & physical)
Create assignments & quizzes
Grade student submissions
Track student performance
Send announcements
Chat with students

🧑‍💼 Institute Officer Features

Verify offline payments
Manage student enrollments
Mark attendance (QR system)
Issue student ID cards
Generate reports

👑 Admin Features

Manage all users (students, teachers, officers)
Approve courses & study materials
Monitor system activity
Generate reports
Manage system backups
Control AI chatbot

🤖 AI Chatbot

Answer student questions
Provide course details
Guide new users

🏗️ System Architecture

This system is built using a modern web-based architecture:
Frontend → React.js (User Interface)
Backend → Node.js + Express
Database → (Supabase / PostgreSQL or MongoDB based on implementation)
Cloud Storage → For study materials
External Services:
Payment Gateway
Email/SMS Notifications
Video Conferencing (Zoom/Meet)
